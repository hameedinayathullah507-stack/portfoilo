# 🌌 Futuristic 3D Developer Portfolio

Welcome to my personal portfolio repository! This project is an immersive, interactive, and fully responsive web experience designed to showcase my skills in frontend development, 3D graphics, and modern UI/UX design.

### 🔗 Live Preview
Check out the live website here: **[Hameed Inayathullah's Portfolio](https://hameedinayathullah507-stack.github.io/portfolio/)**

---

### 🛠️ Technologies Used
- **Three.js**: Powers the interactive 3D particle background and anti-gravity floating objects.
- **GSAP (GreenSock)**: Drives the smooth scroll-triggered animations and dynamic typing effects.
- **Vite**: Used for lightning-fast bundling and local development.
- **Vanilla CSS**: Custom-built styling featuring glassmorphism, responsive grids, CSS clamp() for fluid typography, and dynamic neon drop-shadows.
- **HTML5 & JavaScript**: The core structure, semantic layout, and application logic.

### ✨ Key Features
- **Interactive 3D Environment**: The camera smoothly flies through a space-like scene as the user scrolls down the page, giving a deep sense of parallax and dimension.
- **Dynamic Mouse Parallax**: The entire 3D scene subtly shifts and reacts to the position of the user's cursor.
- **Glassmorphism UI**: Beautiful, frosted-glass interface components combined with glowing neon gradients.
- **Fully Responsive**: Adapts flawlessly to mobile, tablet, and desktop screens using modern CSS layout techniques.
- **Automated Deployment**: CI/CD pipeline set up using GitHub Actions to automatically build and deploy to GitHub Pages on every push to the `main` branch.

### 💻 Running Locally

If you'd like to explore the code or run this project on your own machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/hameedinayathullah507-stack/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the local development server:
   ```bash
   npm run dev
   ```
5. Open your browser and visit `http://localhost:5173`.
