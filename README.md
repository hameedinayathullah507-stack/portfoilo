# 🚀 Hameed Inayathullah — Frontend Developer Portfolio

A **premium futuristic** personal portfolio website built with immersive UI, advanced interactions, and modern frontend design. Crafting digital experiences that feel alive.

### 🔗 Live Preview
👉 **[hameedinayathullah507-stack.github.io/portfolio](https://hameedinayathullah507-stack.github.io/portfolio/)**

---

## ✨ Features

- **Dark Futuristic Design** — Deep dark background with electric blue, purple, and cyan neon accents
- **Glassmorphism UI** — Frosted-glass cards with dynamic inner glow effects
- **Mouse-Follow Glow** — Ambient radial glow that tracks cursor movement across the page
- **3D Animated Background** — Three.js powered particle field and animated grid floor with camera parallax
- **Magnetic Buttons** — CTA buttons that subtly attract toward the mouse on hover
- **Tilt Cards** — 3D perspective tilt effect on project and skill cards
- **Scroll Reveal Animations** — Elements gracefully fade and slide in as you scroll
- **Typing Effect** — Animated character-by-character text reveal in the hero section
- **Sticky Glassmorphism Navbar** — Transparent on load, frosted-glass on scroll
- **Fully Responsive** — Optimized for mobile, tablet, and desktop screens

- **Google Fonts** — Space Grotesk & Inter typography
- **Font Awesome** — Icon library

## 💻 Running Locally

```bash
# 1. Clone the repository
git clone https://github.com/hameedinayathullah507-stack/portfolio.git

# 2. Navigate to the project directory
cd portfolio

# 3. Install dependencies
npm install

# 4. Start the dev server
npm run dev
```

Open your browser at `http://localhost:5173/portfolio/`
